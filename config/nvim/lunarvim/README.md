# LunarVIM

## Requirements

- [sql-language-server](https://github.com/joe-re/sql-language-server)
- [deno](https://docs.deno.com/runtime/manual/getting_started/installation)

## Plugins

You can see all plugins in use in the [`config.lua` file](./config.lua).

- [neotest](https://github.com/nvim-neotest/neotest?tab=readme-ov-file#installation)
- [coc-pyright](https://github.com/fannheyward/coc-pyright)
- [coc-tsserver](https://github.com/neoclide/coc-tsserver)
- [coc-html](https://github.com/neoclide/coc-html)
- [coc-css](https://github.com/neoclide/coc-css)
- [coc-markdownlint](https://github.com/fannheyward/coc-markdownlint)
- [vim-test](https://github.com/vim-test/vim-test)
- [peek.nvim](https://github.com/toppair/peek.nvim)
